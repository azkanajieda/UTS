Esai Materi

Tulisan ini berisi penjelasan mengenai materi yang sudah Saya pelajari di mata kuliah SIG berbasis Web dari minggu pertama hingga minggu ke-7. Mata kuliah Praktikum SIG Berbasis Web membahas mengenai bagaimana data dan informasi geospasial ditampilkan melalui internet, baik pada platform berupa desktop, mobile maupun media lainnya. Mata kuliah ini dimulai dari pembahasan mengenai cara kerja internet dan halaman web, konsep dan cara kerja client dan server pada WebGIS, arsitektur WebGIS serta desain dan evaluasi WebGIS. Praktikum SIG Berbasis Web merupakan salah satu mata kuliah pilihan pada jurusan Teknik Geodesi Universitas Gadjah Mada.

Untuk informasi lebih lanjut mengenai Teknik Geodesi, please kindly check the website on :

Website resmi Teknik Geodesi UGM

Minggu ke-1

Judul atau Tema materi adalah "Pengantar SIG Berbasis Web" dimana mempelajari mengenai Mengenal cara kerja internet dan istilah-istilah internet Mengenal komponen web: HTML, CSS dan JS Latihan membuat web sederhana dengan HTML dan CSS Membaca panduan praktik pembuatan web dengan WSL dan Linux.



Minggu ke-2

Judul atau Tema materi adalah "HTML dan CSS" yang mempelajari lebih jauh mengenai HTML dan CSS serta penerapannya pada pembuatan halaman Web. Pada minggu ini juga diperkenalkan mengenai Flexbox sebagai perangkat penyusunan layout yang banyak digunakan saat ini Praktikum untuk minggu kedua ini: - Latihan Pemahaman HTML dan CSS - Melakukan pembuatan Halaman HTML dan CSS - Menyelesaikan soal-soal dan diskusi terkait CSS.



Minggu ke-3

Judul atau Tema materi adalah "Javascript sebagai Bahasa Client-Side Scripting" Minggu ini merupakan materi kelanjutan dari minggu sebelumnya mengenai Javascript. Pada minggu ini dipelajari peranan Javascript sebagai bahasa pemrograman klien (Client-side scripting) serta dasar-dasar AJAX dan perkembangan Web Modern Praktikum pada minggu ini: - Melanjutkan praktek Javascript - Latihan Mandiri: Game Javascript.



Minggu ke-4

Judul atau Tema materi adalah "Bootstrap Framework" Minggu 4 diisi dengan pembahasan mengenai Boostrap sebagai salah satu framework javascript yang paling banyak digunakan Praktek minggu ini: - Membuat Halaman dengan Bootstrap V4 atau V5 - Kustomisasi Bootstrap dengan Pingendo.



Minggu ke-5

Judul atau Tema materi adalah "Leaflet JS" Minggu ini membahas mengenai salah satu webMap API, yaitu LeafletJS. Praktikum pada minggu ini: - Membuat halaman web sederhana pada LeafletJS - Layouting halaman web dengan Bootstrap - Buat layout peta Leaflet dengan Navbar Bootstrap - Buat sebuah tombol pada navbar yang apabila user mengklik tombol tersebut, maka peta akan zoom ke lokasi user saat ini.Panduan: https://leafletjs.com/examples/mobile/



Minggu ke-6

Judul atau Tema materi adalah "Leaflet JS Lanjut" Minggu ini membahas mengenai LeafletJS, yaitu penambahan layer, pengenalan format GeoJSON dan instalasi Plugin. Praktikum minggu ini: - Tambahkan plugin ke dalam leaflet untuk memperkaya fungsi webgis. plugin yang bisa ditambahkan (misalnya Leaflet Geolocation, Geocoding, Leaflet Graphic Scale, dll) dapat dilihat pada http://leafletjs.com/plugins.html - Gunakan QGIS atau GeoJSON.io untuk mengkonversi file Jogja.shp menjadi GeoJSON, kemudian tambahkan ke kerangka peta yang sudah dibuat. Gunakan panduan: https://leafletjs.com/examples/geojson/ - Mengkombinasikan bootstraps dengan leaflet yang dibuat (geojson + min 1 plugin) untuk membuat sebuah halaman web - Buat halaman web yang serupa dengan menggunakan QGIS2Web.



Minggu ke-7

Judul atau Tema materi adalah "Mobile GIS dengan Input dan Mergin" Pada modul ini akan dibahas mengenai bagaimana aplikasi Mergin Maps: QGIS in pocket sebagai perangkat berbasis mobile, QGIS sebagai perangkat desktop serta Mergin sebagai aplikasi GIS berbasis online dapat digunakan sebagai sarana untuk pengumpulan data dari lapangan secara kolaboratif. Pengumpulan data menggunakan kombinasi ketiga perangkat tersebut dapat digunakan dalam kegiatan survei lapangan, misalnya terkait dengan survei pelanggaran tata ruang, pemetaan aset, pemetaan tempat makan, pemetaan ATM maupun monitoring pembangunan. Pada Praktikum Minggu ke 7 ini, mahasiswa secara berkelompok yang terdiri dari maks 5 orang diminta untuk merancang sebuah kegiatan survei berupa Titik, Garis, Poligon dengan total seluruhnya minimal 30 fitur menggunakan kombinasi ketiga perangkat tersebut. Hasilnya dibuat dalam bentuk laporan dan dikumpulkan terakhir satu bulan dari sekarang.
